# Weather-Forecasting-Application

1. Developed a modern, responsive weather application using Jetpack Compose with a Center-Aligned Material 3 UI.

2. Integrated OpenWeatherMap API using Retrofit to fetch and display real-time weather data and 5-day forecasts.

3. Implemented MVVM Architecture to ensure clean separation of concerns and maintainable code.

4. Applied advanced UI techniques including custom theme colors (BackgroundYellow) and State Management for seamless data updates.
